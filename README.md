# 10.1D
This app is a LLM enhanced study app. It allows for generation of quizes based off a users interests. It uses a database (Room DB) to store the data. App uses remote inference so you will need to run the server package.
As such the following packages:
- Ollama 
- requests 
- flask 
Web server also allows for you to share your profile. also you can display your history to review your answers and finally you can upgrade to premium account using google pay. (please add a valid merchant ID to get this to work)
